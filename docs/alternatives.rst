Alternatives to Scaffolder
==========================

As mentioned in the introduction of these docs, Scaffolder is a reasonably heavy-weight solution for `Node.js`_ CLI applications.  Additionally, it likes to do things a certain way.

There are lots of alternatives to Scaffolder, however, and a few of these are listed below:

- `commander <https://github.com/visionmedia/commander>`_
- `nopt <https://github.com/isaacs/nopt>`_
